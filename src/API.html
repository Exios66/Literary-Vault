Literary Vault API Documentation

Base URL: http://localhost:8000/api/v1

Available Categories

GET /categories

Get all available question categories

curl "http://localhost:8000/api/v1/categories"

Response

{
  "categories": [
    "astronomy",
    "literature",
    "mathematics",
    "psychology",
    "american-history"
  ]
}

Random Questions

GET /questions/{category}/random

Get random questions with optional filters

Parameters

Name Type Description Default
count integer Number of questions (1-20) 5
seed integer Random seed for reproducible results null
topic string Filter by topic null
difficulty integer Filter by difficulty (0-3) null

Examples

// Basic random questions
curl "http://localhost:8000/api/v1/questions/astronomy/random?count=5"

// With topic filter
curl "http://localhost:8000/api/v1/questions/astronomy/random?topic=Planets&count=3"

// With difficulty filter
curl "http://localhost:8000/api/v1/questions/mathematics/random?difficulty=2&count=4"

// Combined filters
curl "http://localhost:8000/api/v1/questions/psychology/random?topic=Cognition&difficulty=1&count=3&seed=12345"

Category Topics

GET /questions/{category}/topics

Get all available topics for a category

curl "http://localhost:8000/api/v1/questions/astronomy/topics"

Response

Category Statistics

GET /questions/{category}/stats

Get statistics about questions in a category

curl "http://localhost:8000/api/v1/questions/literature/stats"

Programming Examples

Python

import requests

base_url = "http://localhost:8000/api/v1"

# Get random questions
response = requests.get(f"{base_url}/questions/psychology/random", 
    params={
        "count": 3,
        "topic": "Cognition",
        "difficulty": 1,
        "seed": 12345
    }
)

# Get category stats
stats = requests.get(f"{base_url}/questions/mathematics/stats")

# Get available topics
topics = requests.get(f"{base_url}/questions/literature/topics")

JavaScript

const baseUrl = "http://localhost:8000/api/v1";

// Get random questions
fetch(`${baseUrl}/questions/american-history/random?count=5&topic=Civil%20War`)
    .then(response => response.json())
    .then(data => console.log(data));

// Get category stats
fetch(`${baseUrl}/questions/psychology/stats`)
    .then(response => response.json())
    .then(data => console.log(data));
